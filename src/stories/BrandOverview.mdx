# Attorney Assistant Brand Guide

This Storybook documents the complete brand system for Attorney Assistant's marketing website.

## Brand Identity

Attorney Assistant provides expert legal support services including Intake 360 and Impact Assistants to help law firms streamline operations.

## Contents

- **Colors** — Brand color palette with hex values, usage guidelines, and accessibility contrast ratios
- **Typography** — Font families, weights, and heading/body hierarchy
- **Buttons** — Primary, secondary, and variant button styles
- **Cards** — Blog cards, testimonial cards, and feature cards
- **Hero** — Hero banner variations
- **CTA Banner** — Call-to-action section variants
- **Logo** — Logo usage and placement guidelines
- **Icons** — Icon set used across the site

## Design Tokens

All design tokens are defined in `tailwind.config.mjs` and available via Tailwind utility classes:

| Token | Tailwind Class | Value |
|-------|---------------|-------|
| Brand Black | `text-brand-black`, `bg-brand-black` | `#0b0000` |
| Brand White | `text-brand-white`, `bg-brand-white` | `#ffffff` |
| Brand Gold | `text-brand-gold`, `bg-brand-gold` | `#ffaa2b` |
| Brand Blue | `text-brand-blue`, `bg-brand-blue` | `#50a7dd` |
| Brand Steel | `text-brand-steel`, `bg-brand-steel` | `#588aa5` |
| Heading Font | `font-heading` | Poppins |
| Body Font | `font-body` | Roboto |
